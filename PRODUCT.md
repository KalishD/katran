# Product

## Register

product

## Users

B2B-закупщики промышленных предприятий: судостроительные заводы, нефтегазовые компании, машиностроительные производства. Ищут конкретный пневмоинструмент по характеристикам, сравнивают аналоги, закупают партиями. Контекст использования: рабочий стол, поиск по артикулу или названию, быстрый переход к характеристикам и цене.

## Product Purpose

Интернет-магазин пневматического инструмента для B2B-аудитории. Помочь закупщику найти нужный инструмент по характеристикам, убедиться в наличии и цене, оформить заказ. Успех — когда покупатель находит товар за < 2 кликов от каталога и оформляет заказ без вопросов менеджеру.

## Brand Personality

Промышленность, практичность, сервис. Голос бренда: уверенный эксперт, который знает инструмент и помогает выбрать, а не навязывает. Тон — деловой, но не холодный; конкретный, без маркетингового шума.

## Anti-references

- Яркие SaaS-лендинги с градиентами, фиолетовым, glassmorphism — слишком «стартап», не для B2B-закупок
- Премиум-минимализм (Apple-like) — слишком дорого, не для промышленного каталога
- Шумные маркетплейсы (Ozon, Wildberries) — перегружены, сложно найти нужное

## Design Principles

1. **Информация > декор** — каждая единица интерфейса должна помогать принять решение о покупке
2. **Экспертная навигация** — закупщик должен найти товар по артикулу, характеристикам или категории, не перекапываясь
3. **Надёжность через стабильность** — интерфейс не должен «прыгать»; предсказуемость = доверие
4. **Сервис в деталях** — быстрый доступ к цене, наличию, характеристикам, аналогам, документации
5. **Промышленная эстетика** — чисто, функционально, без визуального шума

## Accessibility & Inclusion

WCAG AAA. Контраст ≥ 7:1 для body text, ≥ 4.5:1 для large text. Keyboard-navigable каталог. Alt-тексты для всех изображений товаров. Reduced motion для анимаций. Семантический HTML (table для характеристик, proper heading hierarchy).
