# Plan: Add srcset to All Product Image Templates

## Goal
Add responsive `srcset` + `sizes` attributes to every `<img>` tag that renders product/category/brand/post images, so the browser downloads the appropriately sized image for the viewport.

---

## Approach

### Backend: Generate multiple image sizes on save

Extend `ImageProcessingMixin` (`apps/store/models.py:15-66`) to generate 3 variants on every image upload:

| Variant | Max width | Filename pattern | Purpose |
|---|---|---|---|
| `sm` | 400px | `{slug}_sm.jpg` | Mobile cards, thumbnails |
| `md` | 800px | `{slug}_md.jpg` | Tablet, category grids |
| `full` | 1200px | `{slug}.jpg` | Desktop, product detail |

Keep existing 60x60 `thumbnail` for nav/header. All variants are JPEG quality=80, RGB.

### Backend: Template tag for srcset

Create `{% responsive_img %}` tag in `apps/store/templatetags/store_extras.py`:

```django
{% load store_extras %}
{% responsive_img product.image "Product title" css_class="image" loading="lazy" %}
```

Outputs:
```html
<img
  src="/media/uploads/products/slug.jpg"
  srcset="/media/uploads/products/slug_sm.jpg 400w,
          /media/uploads/products/slug_md.jpg 800w,
          /media/uploads/products/slug.jpg 1200w"
  sizes="(max-width: 600px) 100vw, (max-width: 1024px) 50vw, 33vw"
  alt="Product title"
  class="image"
  loading="lazy"
>
```

The tag accepts an ImageField or FileField, reads `.url`, constructs srcset URLs by inserting `_sm`/`_md` suffixes before the extension. Falls back to single `src` if variants don't exist.

### Backend: API JSON responses

Update all views that serialize product data for Vue templates to include srcset URLs:

**`apps/store/views.py`** (3 locations: `category_products_api` ~line 73, `brand_products_api` ~line 226, `search_api` ~line 406):
```python
products.append({
    ...
    'image': p.image.url if p.image else '',
    'image_sm': p.image_sm.url if hasattr(p, 'image_sm') and p.image_sm else p.image.url if p.image else '',
    'image_md': p.image_md.url if hasattr(p, 'image_md') and p.image_md else p.image.url if p.image else '',
    'brand_image': p.brand.image.url if p.brand and p.brand.image else None,
    ...
})
```

**`apps/cart/views.py`** (line 24) — same pattern for cart products.

**`apps/comparison/views.py`** (line 22) — same pattern for comparison products.

### Frontend: Vue template srcset binding

In Vue-bound `<img>` tags, bind srcset dynamically:

```html
<img
  :src="product.image"
  :srcset="`${product.image_sm} 400w, ${product.image_md} 800w, ${product.image} 1200w`"
  sizes="(max-width: 600px) 100vw, (max-width: 1024px) 50vw, 33vw"
  :alt="product.title"
>
```

### Frontend: Django template srcset (non-Vue)

Use the `{% responsive_img %}` tag everywhere else.
